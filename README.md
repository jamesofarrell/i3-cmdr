# i3-cmdr
Used to control i3wm. Requires [i3ipc-python](https://github.com/acrisci/i3ipc-python/).

## usage

### dwindle
`i3-cmdr dwindle (h r|l) | (v u|d)`  
h: horiontal split   
 - l: left to right  
 - r: right to left

v: vertical split  
 - u: top to bottom  
 - d: bottom to top

### float all windows
`i3-cmdr float`
